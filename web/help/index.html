Keyboard Shortcuts

Throughout the application keyboard shortcuts have been implemented as a means to interact with the interface without the need of a pointing device. This ensures that the application is accessible to a diverse audience. The following keyboard shortcuts have been embedded into the system -

CTRL+F - Focuses on the search field, allowing for keywords to be searched.

CTRL+D - Redirects back to the main dashboard.

CTRL+M - Toggles the sidebar menu, allowing access to the different pages.

CTRL+U - Loads the unit section with unit information, and the ability to add a new unit.

CTRL+H - Loads the help section with informative information about the system.

CTRL+W - Adds a new week to the weeks column on the main dashboard.

CTRL+I - Toggles the item menu, listing the different types of items that can be created.

CTRL+L - Logouts of the system, redirects back to the landing page.

Main Dashboard

The main dashboard is the first interface that is displayed after logging in to the system. It is separated into two parts, the weeks column and the items section. The weeks column displays the number of weeks that was created on unit creation, and any new weeks created by the user clicking 'Add Week'. These weeks can be expanded and collapsed to only have viewable the weeks needed.

The items section contains all of the items that has been created by the user, this includes topics, notes, resources, lecturers and rooms. Every item (apart from resources) has a text field where text content can be added, with every item automatically saving the data added to it as the user types. The resource item includes a file upload service, where the user can upload a file / resource to be accessible on the server.

Items can be dragged and dropped from the items section to any of the weeks and vice-versa. Multiple items can be dragged into a single week, allowing lecturers to plan their weeks accordingly.

Units

The units page is accessible from the sidebar by clicking on the menu icon in the top left, or by CTRL+U with the keyboard to go directly to the page. The Change Unit section allows a user to switch between the units that they have created, and to be able to view an overview of each unit, with its creation date, number of weeks, the number of each different type of item, and number of files displayed next to the units. The interface will also list every file uploaded to the unit, with links given to each file.

Also available on Units is the ability for a user to create and add a new unit to their account by simply typing in a name for the unit, and the number of weeks that they require, and clicking 'Add'.

Search

Located within the title bar the user has access to a search field, and by entering in keywords the search algorithm will match these with the best results and display them to the user. Additionally, the user has access to view the type of items they wish to look at, either by searching for the item type, e.g. 'notes', or by selecting the item type page from the sidebar, e.g. 'All Lecturers'.
