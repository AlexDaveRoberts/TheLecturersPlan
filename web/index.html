Items View - View all of the items that you have created previously, and be able to create new items to be dragged and dropped into weeks. Files can also be uploaded to the server as resources.

Weeks View - View all of the teaching weeks for the year in a scrollable column, with the option to expand and collapse weeks to view only the weeks you want, and having every week contain the content that you require.

Units Section - View all of the units associated with your account, and be able to change to a unit you wish to view. Unit information is listed next to each unit, giving you an overview at a glance.
